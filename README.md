# Shitpost Hero

An idle typing game where you build your social media empire one post at a time.

## Play

Visit [marbinner.github.io/idle-typer](https://marbinner.github.io/idle-typer) to play.

## Features

- **Type to earn** - Complete posts to earn coins and followers
- **Idle progression** - Purchase bots that generate coins automatically
- **Upgrades** - Boost your earnings with various multipliers
- **Golden characters** - Hit golden letters for bonus rewards
- **Viral posts** - Get lucky and watch your engagement explode
- **Rank up** - Progress from Lurker to Media Empire and beyond

## Development

```bash

python -m http.server 8080

# Visit http://localhost:8080
```

## Tech Stack

- Vanilla JavaScript (ES6 modules)
- CSS3 with custom properties
- Canvas-based particle effects
- LocalStorage for save data
